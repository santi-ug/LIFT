# LIFT 
A mobile workout tracking app inspired by Hevy for the gym.

**Frontend**: https://github.com/santi-ug/LIFT  
**Backend**: https://github.com/santi-ug/LIFT-backend

&emsp;

To run, clone the repo and follow these steps


**Install Dependencies**
```
npm install
```

**Run server**
```
npx expo start
```

**Scan QR Code on phone**  
_Make sure to have Expo app installed._
